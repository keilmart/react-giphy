## Running The App

### `git clone https://github.com/keilmart/react-giphy.git`

First Clone the Repo

-

### `cd react-giphy`
### `npm install`

Install the dependancies

-

### `npm start`

Start the App!

-

### `http://localhost:3000/`

If the app dosent auto open, open localhost:3000

-

### `npm test`

Launches the test runner in the interactive watch mode.\
